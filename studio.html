Import / Export

Работа с JSON

Поддерживается массив задач или объект вида { "tasks": [...] }.

Изменения сохраняются локально в браузере и сразу становятся видны на ученической странице.
Task Schema

Минимальная структура задачи.

  • title Короткое название.
  • category Тематика: последовательности, стратегия и так далее.
  • difficulty Старт, Уверенно или Сложно.
  • prompt Формулировка задания.
  • visual Массив элементов или строка через запятую.
  • options Варианты ответа.
  • answer Индекс правильного варианта с нуля.
Secure Entry

Доступ к библиотеке автора задач.
