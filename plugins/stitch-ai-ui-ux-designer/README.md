# Stitch AI UI/UX Designer

Local Codex plugin for UI/UX design work inside this repository.

## What it does

- Turns rough product ideas into concrete screen and flow direction
- Reviews interfaces for UX, hierarchy, and accessibility issues
- Helps translate design intent into implementation-ready frontend work
- Preserves existing product patterns when working inside an established codebase

- Skill: `skills/stitch-ai-ui-ux-designer/SKILL.md`
